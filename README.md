# Selenium Java Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Apache POI
- Log4j
- Git & GitHub

## Features
- Page Object Model
- Data Driven Testing using Excel
- Screenshot Capture on Failure
- Logging using Log4j
- TestNG Reporting

## Project Structure
src/main/java
src/test/java
reports
screenshots
logs

## Execution
mvn test
