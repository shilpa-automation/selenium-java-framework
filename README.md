# Selenium Java Automation Framework

## Overview
This project is a Selenium Automation Framework developed using Java, Maven, and TestNG. It follows the Page Object Model (POM) design pattern and supports data-driven testing.

## Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- Git & GitHub
- Apache POI (Excel Data Handling)
- Log4j

## Framework Features
- Page Object Model (POM)
- Data-Driven Testing using Excel
- Reusable Utility Methods
- TestNG Test Execution
- Logging Support
- Screenshot Capture on Failure
- Maven Build Management

## Project Structure

src

## Prerequisites
- Java 8 or above
- Maven
- Eclipse/IntelliJ IDEA
- Chrome Browser

## Execute Tests

```bash
mvn clean test
```

```bash
testng.xml
```
